Array Methods

  • Push method = is for mag add sa dulo ng array
  • Pop Method = mag tanggal sa dulo
  • Unshift Method = mag add sa unahan
  • Splice Method = ay ginagamit para magtanggal, magdagdag, o magpalit ng item sa array sa kahit anong posisyon.
    1. Remove array.splice(startIndex from 0, startCount from 1);
    2. Insert ang splice people.splice(1, 0, "B");
    3. Mag Add gamit ang splice at mag replace people.splice(1, 1, "B");
    4. Insert Multiple Items people.splice(1, 0, "B","A","C");
    5. Insert and replace Multiple Items people.splice(0, 3, "B","A","C");
