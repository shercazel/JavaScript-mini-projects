Array Methods

  • Push method = is for mag add sa dulo ng array
  • Pop Method = mag tanggal sa dulo
  • Unshift Method = mag add sa unahan
  • Splice Method = ay ginagamit para magtanggal, magdagdag, o magpalit ng item sa array sa kahit anong posisyon.
    array.splice(startIndex from 0, startCount from 1);
